Width and Height in CSS

            
Here are three common ways to target and style elements.


1. **ID Selector (`#`)**  
Targets a single element with a unique ID. Use `#idName` in CSS to style the element with `id="idName"`.  
Example:  
```css
#header {
  background-color: blue;
}
```
```html
This is the header
```

2. **Class Selector (`.`)**  
Targets one or more elements with a shared class. Use `.className` in CSS to style elements with `class="className"`.  
Example:  
```css
.highlight {
  color: red;
}
```
```html
This text is red.
This text is also red.
```

3. **Element Selector (`div`)**  
Targets all elements of a specific type. Use the element name directly to style all such elements.  
Example:  
```css
div {
  padding: 10px;
  border: 1px solid black;
}
```
```html
All div elements get padding and border.
Another div.
```

***

### Summary:

| Selector       | Syntax   | Targets                          | Example         |
|----------------|----------|---------------------------------|-----------------|
| ID selector    | `#id`    | Single element by unique ID      | `#header`       |
| Class selector | `.class` | One or many elements by class    | `.highlight`    |
| Element selector | `element` | All elements of that type (e.g., `div`, `p`) | `div`          |

Each method helps you control styling at different levels of specificity and reusability.

Size in CSS

In CSS, size usually refers to controlling the width and height of elements. Here’s a quick overview.

Common Value Types

units: px (pixels), em, rem, cm etc.
Example: width: 500px;

Percentage: relative to the size of the containing element.
Example: width: 50%;

auto: lets the browser calculate the size (default).

Additional Related Properties max-width and max-height: sets maximum size limits. min-width and min-height: sets minimum size limits.

Min-Max Width in CSS

In CSS, min-width and max-width are properties used to control the minimum and maximum width of an element. They are very helpful for making responsive designs.

min-width

Specifies the minimum width of an element.;

The element’s width will never be smaller than this value, even if its content or container is smaller.;

Useful for preventing elements from becoming too narrow.

Additional Related Properties max-width and max-height: sets maximum size limits. min-width and min-height: sets minimum size limits.
